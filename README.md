# khuthon 2024 - 팀AID 서버

## 대회
- 일시: 2024.05.10 ~ 2024.05.11
- 장소: 경희대학교 국제캠퍼스 전자정보공학관
- 주최: 경희대학교 소프트웨어교육센터

## 팀
- 팀명: AID
- 소속: 부산대학교 인공지능 동아리 AID
  - 김민석
  - 이동훈
  - 박준혁
  - 하석현

## 프로젝트 개요
- 주제: 환경과 소프트웨어
- 프로젝트명: 컴퓨팅 자원을 효율화시켜주는 데스크탑 AI 비서

- BE: https://github.com/KimCookieYa/2024_Khuthon_AID_server
